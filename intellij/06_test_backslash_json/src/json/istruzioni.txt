Per creare la versione di "parametri-template.json" specifica per la macchina attuale, creare una copia di "parametri-template.json" e chiamarla "parametri.json", dopodiché modificare il valore della chiave "percorso_ccs_da_analizzare", impostandolo sulla cartella che contiene i .ccs. Nel caso in cui ci si trovi in Windows, i backslash devono essere sostituiti con doppi backslash.
Inoltre, modificare il valore di "nome_dir_file_batch", impostandolo su una cartella in cui il software potrà accedere per svolgere operazioni sui file batch.
Per quanto riguarda la chiave "percorso_cwb", il percorso tipico in Windows è:
"C:\\CWB-NC\\bin\\cwb-nc.bat"
